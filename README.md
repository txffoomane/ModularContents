# ModularContents

Мод для **Minecraft Forge 1.12.2**, позволяющий создавать собственный контент —
предметы, блоки, еду, рецепты, вкладки креатива, лут и механики аирдропов —
через JSON-файлы или внутриигровой визуальный редактор, **без знания Java**.

- **Версия:** 0.3-Alpha
- **Minecraft:** 1.12.2
- **Discord:** https://discord.gg/JWV6A4JdTN

Подробное руководство по созданию контента: [`ModularContents_Documentation.md`](ModularContents_Documentation.md)

## 1. Что уже реализовано

### Контент-паки
- Система контент-паков в папке `ModularContents/` — неограниченное число паков, каждый со своей структурой (`items/`, `recipes/`, `tabs/`, `loot_tables/`, `blocks/`, `food/`, `assets/`).
- Загрузка кастомных ресурсов (текстуры, модели, языковые файлы) через встроенный `ModularResourcePack`.
- Синхронизация контента и требуемых паков между сервером и клиентом при входе игрока.

### Внутриигровой редактор (Content Creator, клавиша F8)
- Визуальный генератор JSON с разделами: **Items, Blocks, Food, Loot, Recipes, Tabs, Zone, NPC**.
- Прокручиваемая панель вкладок, редактируемые поля и предпросмотр слотов.

### Кастомный контент из JSON
- **Предметы** (`ItemCustom` / `CustomItemInfo`): имя, размер стака, прочность, вкладка креатива.
- **Блоки** (`BlockCustom` / `CustomBlockInfo`): прочность, сопротивление, уровень света, инструмент, материал, уровень добычи.
- **Еда** (`ItemCustomFood` / `CustomFoodInfo`): восстановление голода, насыщение, флаг мяса, «всегда съедобно», зельевые эффекты.
- **Вкладки креатива** (`CustomTabManager`) с предметом-иконкой.

### Крафт
- **Кастомный верстак** (`ListWorkbench`) — многоблочная структура с TileEntity и собственным GUI.
- Рецепты из JSON с вероятностным выходом, несколькими результатами, временем крафта и поддержкой NBT.
- Рецепт крафта самого верстака настраивается через конфиг.

### Аирдропы
- Блок и сущность аирдропа (`EntityAirdrop`) с падением, дымом (частицы + звук) и лутом.
- Вызыватели аирдропов: **рация** (`ItemRadio`) и **сигнальная ракета** (`ItemSignalFlare`), настраиваемые через блок `airdrop_caller` в JSON предмета.
- JSON лут-таблицы для аирдропов (`AirdropLootManager`) с диапазоном количества и шансами.

### Зоны и заполнение сундуков
- **Equipment Manager** с картой-радаром (256×256) для выделения зон и просмотра контейнеров.
- Заполнение сундуков случайным лутом по пресетам (`loot_tables/equipment/`) с наборами и шансами.
- Зоны респавна лута сохраняются в мире и восстанавливают лут по таймеру (`LootZoneManager`).

### NPC
- Кастомные NPC (`EntityCustomNPC`) из JSON: здоровье, скорость, урон, дальность преследования/стрельбы, текстура, экипировка.
- Снаряды NPC (`EntityNPCBullet`) для дальнего боя.

### Прочее
- Команды оператора: `/mc reload` (горячая перезагрузка контента) и `/mc airdrop <x> <z> [loot_table]`.
- Конфигурация мода (`ModularContentsConfig`), настраиваемая привязка клавиш (`KeybindManager`).
- Утилиты упаковки паков (`PackZipUtils`) для доставки требуемых паков клиентам.
